I love using `ncu --format group` to check one what package needs updates. I know many folks love using the built in upgrade interactive, but it's just a little too much for me for an at-a-glance look. What if you do the unthinkable and use yarn@3? This is a good option.

Grab `ncu` from `npm i -g npm-check-updates`.
